# DiscordTool
簡単にWebhookのスパムや一斉参加、退出等の基本動作ができます。

# 特徴
- 軽量
- 多機能
- わかりやすいUI
- Proxyをサポート

# 一斉参加の招待コード
NG例

```
discord.gg/example
```
OK例
```
example
```
「discord.gg/」以降のランダムなアルファベットのコード。

# できること
- Join flood
- Left flood
- Spam
- Webhook spam
- Token info view
- Nitro generate
- Nitro active check
